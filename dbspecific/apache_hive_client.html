Aqua Data Studio for

Enhanced in 17.0 Aqua Data Studio provides a management tool for the Apache Hive[1] and Spark [2] open source distributed database management system with administration capabilities and a database query tool. The visual administration features provide users the ability to browse and modify database structures, including schema objects such as Tables, Indexes and Views. An integrated query tool allows you to quickly create, edit and execute SQL queries and scripts. Aqua Data Studio further provides export tools to move data easily out of the Apache Hive and Spark database.
