Custom Business Intelligence Dashboards
Aqua Data Server

Using the Aqua Data Server charting and reporting library, you can build and display visual reports based on datasets retrieved from heterogeneous data sources within Aqua Data Server scripts. In this way, each user within your organization can benefit from a personal dashboard—a customized and dynamic view of the reporting page. Providing users with a dashboard eliminates the need to build new reports for every occasion; each user can create a custom report at any time. Aqua Open APIs offer a variety of GUI items that can be placed on the dashboard along with the report and the graph. Dashboard users can then run reports by entering just a few parameters, such as date ranges, value thresholds, selected values from a list of values, etc. They can save, export or email reports directly from the dashboard as TXT, HTML, XML or MS Excel files.



Use these Aqua Data Server packages:

aquafold.openapi.chart
aquafold.openapi.report
aquafold.openapi.form to generate and render charts and reports.
