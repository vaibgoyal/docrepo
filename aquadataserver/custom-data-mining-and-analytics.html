Custom Data Mining and Analytics
Aqua Data Server

Aqua Data Server allows you to build solutions for querying and performing complex analyses of the data stored in databases. Aqua Open APIs provide methods for accessing databases of all major vendors, including: Oracle, IBM DB2, MS SQL, Sybase, MySQL, Informix, Netezza, Teradata, Teradata Aster, Sybase and Apache Derby. Generic ODBC implementation provides access to other data sources. The Aqua Open APIs' math package includes various statistical, financial, algebraic and logical functions, which may be used for data analysis. Data processing algorithms are implemented in the form of AquaScripts—a JavaScript based programming language—and deployed on Aqua Data Server for access and execution either on-demand by users, or on a schedule through the built-in Job Scheduler. You can display processed datasets as tables and graphs directly within the Aqua Data Server web-based interface. Aqua Open APIs also provide a variety of functions for building GUI elements. Datasets may be exported as TXT, HTML, XML or MS Excel files, and saved on the local file system.



Use these Aqua Data Server packages:

aquafold.openapi.rdbms and aquafold.openapi.rdbms.schema to access and manage databases of different vendors;
aquafold.openapi.math to implement data processing algorithms;
aquafold.openapi.type to manipulate data of different types.
