Custom Solutions with Open Aqua APIs

Aqua Data Server includes a complete set of APIs for quickly building and deploying custom solutions for data mining, ETL operations, database/file system data exchange, data warehousing and other data processing tasks. Aqua Open APIs are accessible via AquaScript (a JavaScript-based programming language) and include a large number of universal methods for data access, networking, encryption, access security, statistical and math functions. Using Aqua Open APIs helps developers focus on application-specific code and business logic. Aqua Data Server also offers developers a convenient IDE for debugging AquaScript application code, managing database connections and storing application data files.



List of Aqua Data Server Open APIs

  • com.aquafold.openapi.archive - A set of methods and algorithms to archive files
  • com.aquafold.openapi.chart - A set of methods for rendering charts
  • com.aquafold.openapi.compare - A set of methods for comparing files, directories and schemas
  • com.aquafold.openapi.crypto - A set of cryptographic functions
  • com.aquafold.openapi.data - A set of functions for working with data grids and pivot tables
  • com.aquafold.openapi.filesystem - A set of functions for accessing and managing computer file system
  • com.aquafold.openapi.form - A set of methods for implementing HTML forms
  • com.aquafold.openapi.io - A set of methods for reading and writing in TXT, HTML and Excel formats
  • com.aquafold.openapi.mail - A set of methods for sending and receiving e-mails
  • com.aquafold.openapi.math - A set of mathematical and statistical functions
  • com.aquafold.openapi.net - A set of networking methods
  • com.aquafold.openapi.project - A set of methods which abstract the resources in the AquaProject to be access programmatically, and identical in Aqua Data Studio and Aqua Data Server
  • com.aquafold.openapi.random - A set of random data generators
  • com.aquafold.openapi.rdbms - A set of methods for accessing various types of databases
  • com.aquafold.openapi.rdbms.schema - Set of methods for manipulating database objects
  • com.aquafold.openapi.rdbms.util - Set of methods for parsing and formatting SQL statements
  • com.aquafold.openapi.report - A set of methods for database querying and reporting
  • com.aquafold.openapi.type - A set of methods for data manipulation and validation for various data types
  • com.aquafold.openapi.util - A set of convenient methods for development and debugging
  • com.aquafold.openapi.vcs - A set of methods for working with CVS and Subversion version management systems
