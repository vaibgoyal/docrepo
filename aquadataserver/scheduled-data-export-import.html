Scheduled Data Export/Import
Aqua Data Server

With Aqua Data Server, you can access heterogeneous data sources from relational databases of major vendors, including: Oracle, IBM DB2, MS SQL Server, Sybase, MySQL, Informix, Netezza, Teradata, Teradata Aster, Sybase and Apache Derby. Generic ODBC implementation provides access to other data sources. Using AquaScripts—a JavaScript-based programming language—you can build routines for exporting and importing data between databases and the local file system. Aqua Open APIs include functions for accessing local file systems and reading and writing data in the TXT, HTML and Excel file formats. Scripts can be developed and debugged using Aqua Data Studio and deployed to Aqua Data Server. You can even set up your Export/Import scripts to run on a predefined schedule using Aqua Data Server's built-in job scheduler.



Use these Aqua Data Server packages:

aquafold.openapi.io to read and write data from and to TXT, HTML and MS Excel files.
